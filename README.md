# EliasChaosFractal-Generic (Rust Version)

A Rust implementation of the EliasChaosFractal self-evolving fractal gossip node, versioned at 3.2.9 to sync with the Apple (Swift) repository. This crate provides a stress-hardened, recursively self-aware node with distributed simulation and query classification capabilities.

## Features

- **Recursive Intelligence**: Processes queries with adaptive depth and fractal dimension reflection.
- **Stress-Hardened**: Circuit breakers, smoothed fractal dimensions, and throttled query handling.
- **Distributed Simulation**: Simulates 50+ nodes to test network resilience.
- **Query Classification**: Prioritizes queries by complexity for optimized load management.
- **Configurable**: Uses `config.json` for runtime settings like storage tokens and limits.
